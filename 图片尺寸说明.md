# 拼图图片尺寸说明

## 推荐尺寸（每块200x200像素）

### 4块拼图（简单难度）
- **整体尺寸：400 x 400 像素**
- **网格：2列 x 2行**
- **每块尺寸：200 x 200 像素**
- **文件夹：images/4/**

### 6块拼图（中等难度）
- **整体尺寸：600 x 400 像素**
- **网格：3列 x 2行（长方形）**
- **每块尺寸：200 x 200 像素**
- **文件夹：images/6/**

### 9块拼图（困难难度）
- **整体尺寸：600 x 600 像素**
- **网格：3列 x 3行**
- **每块尺寸：200 x 200 像素**
- **文件夹：images/9/**

---

## 高清版本（每块300x300像素）

如果你想生成更高清的图片，可以使用：

### 4块拼图（高清）
- **整体尺寸：600 x 600 像素**
- **每块尺寸：300 x 300 像素**

### 6块拼图（高清）
- **整体尺寸：900 x 600 像素**
- **每块尺寸：300 x 300 像素**

### 9块拼图（高清）
- **整体尺寸：900 x 900 像素**
- **每块尺寸：300 x 300 像素**

---

## 注意事项

1. **图片必须是完整的**：生成的图片应该包含完整的动物/图案，不会被分割成空白块
2. **正方形块**：虽然6块拼图整体是长方形（600x400），但每块拼图块都是正方形（200x200）
3. **内容分布**：确保每一块都有内容，不要出现大块空白区域
4. **文件格式**：推荐使用 PNG 格式，支持透明背景
